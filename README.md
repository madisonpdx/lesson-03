Introduction
------------

A simple web app that sends and receives text messages and makes and answers voice calls.

Demo App
--------

This is a demo application I setup to show some of the things you can do with SMS and voice calls from your own code.

[Demo App](http://fierce-wildwood-5399.herokuapp.com/)

Twilio
------

This is a service that allows your programs to handle SMS and voice calls. It does cost money, but can be very affordable if you have an app that you charge money for.

[Twilio](https://www.twilio.com/)
